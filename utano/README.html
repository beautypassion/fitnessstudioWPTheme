Getting Started

This document serves as an introduction to the utano - Dance & Fitness Studio WordPress Theme. In this guide, you will learn how to install utano WordPress Theme and its Demo Content as well as make basic settings.

Installation

After WordPress installation upload the .zip package with utano WordPress Theme in a themes directory and unzip everything. Also you can upload utano WordPress Theme from your WordPress Dashboard going to Appearance => Themes => Add New Theme => Upload Theme. Then activate it.

Required Plugins

This Theme supports Required Plugins: Advanced Custom Fields (ACF) and One Click Demo Import. Look at the top of your Dashboard you will see notification with relevant links. Just click Begin installing plugins to install them.

DEMO Content Installation

After installation of Required Plugins you will find an Import Demo Data button at the Appearance Admin Menu item. Go to the Manual demo files upload and then upload a .xml File with Demo Content pushing the button under the Choose a XML file for content import element.

A .xml File with Demo Content named DEMO-CONTENT.xml you can find in utano WordPress Theme root folder.

After the Demo Content has been installed go to the Admin Panel Settings => Reading => Your homepage displays and select A static page, then Homepage => Front Page and push Save Changes button.

Social Media Buttons (Footer)

To add Social Media Buttons to the Footer go to the Additional Admin Menu item and fill in relevant fields.

Congratulations! You have finished basic settings. Keep reading to make your own Custom Settings.

Change the Logo

Go to the WordPress Customizer => Site Identity => Select logo and upload file with logo image (.png, .jpg) from your computer. To scale an image navigate to Admin Panel => Media and select uploaded file with logo-image then scale it.

Site Title and Tagline

You can change a Site Title or a Tagline in Customizer => Site Identity as well as in Admin Panel following the path: Settings => General.

Site Icon

To add a Site Icon navigate to Customizer => Site Identity => Site Icon and select image (.png, .jpg) which must be 512 x 512 px.

Navigation Menu

The Navigation Menu at the top you can customize following the Customizer => Menus => Menu 1 and changing Navigation Label field. Also you can go to the Admin Panel and select Appearence => Menus => Menu 1.

Header

utano - Dance & Fitness Studio WordPress Theme supports the Header Image and Header Text Color customization. To set it navigate to Customizer => Header Image or Customizer => Colors.

utano WordPress Theme works best with Header Image size of 1000 x 250 px, cropping of uploaded image is possible.

Site Fields

To customize Title, Backgroung Image or some free text of Site Fields navigate to Post Titles in Admin Panel of your Dashboard. Each field have an unique ID named Title ID, which is responsible for its sequence number. To change a Section Background Image (Memberships and Contact Us) navigate to Post Titles => Memberships or Post Titles => Contact Us and then to the right sidebar.

To set the main information of Site Fields (section items) select a relevant Custom Field Item (Admin Panel Item named as Site Field).

  1. The field "Advantages" contains 3 entries fields. The Icon Class field needs to add a Font Awesome Icon name placed within quotes (f. ex. <i class="fab fa-adn"></i> = fab fa-adn).

  2. The field "Partners" is intended to input partners logos (right sidebar => Featured Image) with a link (Field Link). You also may use it for some alternative text you need.

  3. "Classes" contains information about Fitness Classes with modal windows for a full description. The Class Description field is intended just for short motivational phrase characterizing this class. On the right sidebar you can find a Featured Image field to add a class image to your item.

    Class Aditional Information input field can contain an article about fitness discipline and image, this information will be displayed in modal window.

    The ID Field is used to manage sequence numbers of section elements.

    Notice that you can change the Dumbbell Icon on the yellow background of hovered class element, navigating to Post Titles => Classes => Section Additional Text and entering a Font Awesome Icon name you need to display, placed within quotes (looks like in the field Advantages => Icon Class).

  4. The field "Team" contains an information about fitness coaches: name, speciality, free text about career and achievements, photo (right sidebar) and social media buttons. With an empty coach social media field the social media icon will not be displayed.

  5. The field "Memberships" can store a pricing policy and membership benefits information. Background image and text for the link to Contact Form section you will find navigating to Post Titles => Memberships settings field. Text for the link to Contact Form is placed in the field Section Additional Text.

  6. The "Contact Us" field you can customize navigating to Post Titles => Contact Us. You can add your contact information to the Section Additional Text field. Use a Visual editor to format your text or Text editor to format its HTML markup.

    To set your Contact Form navigate to Admin Panel => Additionally and enter the data: Contact E-mail for sending messages from Contact Form to and E-mail address for generated message for setting an email address the generated message will be from (if your hosting provider require it).

Privacy Policy and Terms of Use for your Website

Using utano - Dance & Fitness Studio WordPress Theme you are able to set Privacy Policy and Terms of Use for your Website in two ways. You can use ready text or add your own to modal window in the footer. To add your own Privacy Policy or Terms of Use text navigate to Admin Panel and select Privacy Policy or Terms of Use menu item. Change the date in Published on item on the right sidebar and it will be displayed in modal window with your Website name and contact email you have entered for sending messages from Contact Form to. Another way is available if you have a HTML Document with your Privacy Policy or Terms of Use text. Put a .html file to the WordPress root Directory and enter a link to it navigating to Admin Panel => Additionally => Link to Privacy Policy page or Link to Terms of Use page. It should look like your_domain_name/file_name.html.

Copyright

To change a Copyright notice go to Admin Panel => Additionally => Copyright and add your own text (f. ex. text &copy year).
